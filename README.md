# Sourish Chattopadhyay - Portfolio Website

A modern, responsive portfolio website for Sourish Chattopadhyay, a professional 3D Modeler and AutoCAD Master.

## 🚀 Features

- **Modern Design**: Clean, professional design with dark theme
- **Responsive**: Fully responsive across all devices
- **Smooth Animations**: Powered by Framer Motion
- **Interactive Portfolio**: Modal gallery for project showcase
- **Contact Form**: Functional contact form with validation
- **SEO Optimized**: Meta tags and proper structure
- **Fast Loading**: Optimized for performance

## 🛠️ Technologies Used

- **React 18** - Frontend framework
- **Framer Motion** - Animations and transitions
- **React Icons** - Icon library
- **CSS3** - Styling with modern features
- **Vercel** - Deployment platform

## 📱 Sections

1. **Hero Section** - Introduction and call-to-action
2. **About** - Professional background and statistics
3. **Skills** - Technical expertise and capabilities
4. **Portfolio** - Project showcase with modal view
5. **Contact** - Contact information and form
6. **Footer** - Additional links and information

## 🚀 Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd sourish-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Building for Production

```bash
npm run build
```

## 🌐 Deployment

### Deploy to Vercel

1. Push your code to GitHub
2. Connect your repository to Vercel
3. Vercel will automatically detect it's a React app and deploy

### Manual Deployment

1. Build the project:
```bash
npm run build
```

2. Deploy the `build` folder to your hosting provider

## 🎨 Customization

### Colors
The website uses a consistent color scheme:
- Primary: `#667eea` (Blue)
- Secondary: `#764ba2` (Purple)
- Background: `#0a0a0a` (Dark)
- Text: `#ffffff` (White)
- Muted: `#b0b0b0` (Light Gray)

### Fonts
- **Poppins** - Headings
- **Inter** - Body text

### Adding Projects
To add new projects to the portfolio, edit the `projects` array in `src/components/Portfolio.js`.
